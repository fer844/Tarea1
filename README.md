# Quién es Más Víbora? 🐍

## Definición del Problema
El desconocimiento sobre las serpientes genera miedos innecesarios y pone en riesgo a diversas especies. **Quién es Más Víbora** es una aplicación web educativa diseñada para enseñar a los usuarios sobre las características únicas de las serpientes, como la Cobra Real y la Serpiente de Coral, permitiendo identificar su nivel de peligrosidad y hábitats de una manera interactiva y visualmente atractiva.

##  Tecnologías Utilizadas
- **React 18**: Para la construcción de la interfaz por componentes.
- **Vite**: Como herramienta de compilación rápida.
- **NPM**: Para la gestión de las librerías del proyecto.
- **CSS3 (Flexbox/Grid)**: Para el diseño responsivo y la organización de las tarjetas de serpientes.

## Pasos de Creación de la App

1.  **Configuración del Entorno:** Se inicializó el proyecto con `npm create vite@latest` para asegurar una base moderna y ligera.
2.  **Arquitectura de Carpetas:** Dentro de la carpeta `src/`, se organizó el proyecto en:
    - `assets/`: Para almacenar las imágenes de las especies.
    - `components/`: Donde se crearon archivos como `Navbar.jsx` y `Card.jsx`.
    - `pages/`: Para la vista principal del proyecto.
3.  **Componentización Dinámica:** Se implementaron componentes funcionales que reciben datos (como el nombre científico y la descripción) para renderizar las tarjetas de la Cobra Real y la Serpiente de Coral de forma dinámica.
4.  **Estilos Profesionales:** Se aplicaron clases de CSS para lograr el encabezado púrpura, el uso de tipografía limpia y badges de peligro (como "Extremadamente Alto").
5.  **Importación de Recursos:** Se gestionaron las imágenes localmente desde la carpeta de activos, utilizando etiquetas semánticas de HTML5 para una mejor estructura.
